# iPadに「アプリっぽく」入れる方法(ホーム画面に追加)

冒険ドリルなどと同じように、ホーム画面のアイコンから直接ひらけるようにする手順です。
アプリストアを使わず、Safari の「ホーム画面に追加」機能で行います(無料・審査不要・すぐ使えます)。

## 手順(iPad・Safari)

1. iPad の **Safari** で以下のURLを開く
   `https://hassy0511.github.io/project01/`
2. 一度、画面の どこかを タップ(音とストーリーが始まればOK。子供が最初にやる分には
   そのまま紙芝居を見せてあげてください)
3. 画面下(または上部)の **共有ボタン**(四角から矢印が上に出ているアイコン 📤)をタップ
4. 出てきたメニューを下にスクロールし、「**ホーム画面に追加**」をタップ
5. 名前はそのまま(「めいさんクエスト」)で右上の「**追加**」をタップ
6. ホーム画面に 🐤 アイコンが追加されます

これで、そのアイコンをタップすると Safari のアドレスバーなどが出ない
**フルスクリーンのアプリのような画面**で起動します。

## 更新について

Web版なので、こちらで機能追加してデプロイするたびに、次にアプリを開いた時に
自動で最新版に更新されます(App Store のような「更新」操作は不要です)。
電波が無い場所でも、一度開いたことがあれば起動できます(オフライン対応ずみ)。

## もし表示がおかしい・新機能が反映されない時

以前お伝えした「キャッシュのクリア」が有効です。
ホーム画面のアイコンを長押し →「**削除**」→ 上の手順1からもう一度「ホーム画面に追加」を
やり直せば、確実に最新版になります(進行データはlocalStorageに残るので消えません。
ただし「データを削除」をSafari側で行った場合は消えるので、それだけは避けてください)。

## 技術メモ(Claude Code 用)

- `public/manifest.webmanifest`: PWA マニフェスト。アイコン・表示名・スタンドアロン表示を定義
- `public/icons/`: `scripts/icon.svg` / `scripts/icon-maskable.svg` を
  `scripts/gen-icons.mjs`(Playwrightでラスタライズ)で生成した各サイズPNG。
  アイコンの絵を変えたい場合は SVG を編集して `node scripts/gen-icons.mjs` を再実行する
- `public/sw.js`: オフラインキャッシュ用サービスワーカー(ビルド出力のハッシュに依存しない
  ランタイムキャッシュ方式。index.htmlは install 時にプリキャッシュ、他は
  stale-while-revalidate)
- `index.html`: `apple-mobile-web-app-*` meta タグ・`apple-touch-icon` 一式・
  `viewport-fit=cover` + `env(safe-area-inset-*)` でノッチ・ホームバー対応
